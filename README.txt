Csapat

MacOs-t használtam. 15-ös jdk-t kellett használjak, hogy a megjelenített programon rendes Times New Roman, UTF-8 -as kódolás menjen.

Root-ban clean,package,install után nekem fut mind az asztali, mind a webes felületem. Project strucutre-ben sdk 15, project language level 11. Szerintem más fontosabb lépés nem kell. 
Asztali alkalmazásunkban tudunk játékost,csapatot hozzáadni, módosítani, törölni vagy akár rájuk szűrni.
Webes felület megnyitása után, tudunk mérkőzést hozzáadni, illetve csapaton belül üzenetet küldeni a játékosunknak. Ha lejátszották a mérkőzést utána eredményt megadni.



Database-t itt található: csapat/csapat-desktop/src/main/resources mappában.
